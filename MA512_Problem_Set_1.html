Exercise 1.1.2
We will show (R, d), where d(x, y) = |x −y|, is a metric space. First, if x, y ∈ R,
then |x − y| ∈ [0, ∞) by deﬁnition of the absolute value function. Taking any
x ∈ R, we ﬁnd d(x, x) = |x − x| = |0| = 0 holds trivially. Now if y ∈ R such
that x = y, then clearly d(x, y) = |x − y| > 0, for |x − y| = 0 only when x = y.
We may also note that d(x, y) = |x −y| = |y −x| = d(y, x) by the properties of
the absolute value function. Finally, if we take an additional z ∈ R, note:
d(x, z ) = |x − z| = |x − y + y − z| ≤ |x − y| + |y − z| = d(x, y) + d(y, z)
Which follows from the triangle inequality as a property of the absolute value
function. These four statements imply that (R, d) satisfy the conditions of Def-
inition 1.1.2, and so (R, d) is a metric space. ■
