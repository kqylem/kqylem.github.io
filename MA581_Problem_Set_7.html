Problem 7.48
To prove that a bounded discrete random variable (bounded by M ) has ﬁnite
expectation and ∣E(X)∣ ≤ M, we ﬁrst observe that by deﬁnition of absolute
value, P (−M ≤ X ≤M )=1. Then, P (X ≥ −M )=P (X +M ≥0)=1, which then
implies that E(X +M) = E(X)+M ≥ 0. Therefore E(X) ≥ −M is validated.
Now, ﬁnd that P (X ≤ M ) = 1, and from before, P (X −M ≤ 0) = P (M −X ≥
0) = 1. This similarly implies that E(M −X) = M −E(X) ≥0, and hence that
M ≥ E(X). Finally, we see that M ≥E(X), and hence M ≥E(X) ≥ −M , which
is simply ∣E(X)∣ ≤ M , as desired.
