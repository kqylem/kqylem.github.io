set V ⊆ E. Then for all x ∈ V we know there exists an r > 0 for which
B(x, r) ⊆ V ⊆ E. This is the deﬁnition of x ∈ int(E), to which V ⊆ int(E).

in the interior of E.
Now suppose we have a closed set K such that E ⊆ K. Take x an adherent
point of E. Then either x ∈ int(E) or x ∈ ∂E. If x ∈ int(E), then clearly
x ∈ K. If x ∈ ∂E, then either x ∈ K or x ∈ ∂K, but since K is closed, this
is equivalent to x ∈ K. Thus in any case, adherent points of E are contained
in K, to which E ⊆ K. Since K was arbitrary, it follows that if any closed set
contains E then this set also contains the closure of E. ■
